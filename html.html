Basics

Here, you will learn about making a basic webste, which includes forms, text styles, heading and many more!

Code explained

This is a basic sample webiste. So, we'll explain how the code works

In every HTML file, all the code must be between <html> tags.

The title of the tab will be marked by the <title> tag.

Headings

The h1tag is the largest. Following <h2>, <h3>, <h4>, <h5>, <h6>, it gets smaller.

Linking and Lists

We use that <a href = '#'> tag is used for links.

The <ol> tag uses numbers to list values, while the <ul> tag uses bulletin points.

We use <li> tags to make lists.

Input and forms

We use the <Form> tag to mark that this is a form and we use <select> tags to mark that there will be choices in this section, which are <option> and <input>.

There are many types of inputs.

The <input> tag is used to create inputs.

Different types of inputs include:

  • <input type = 'text'>
  • <input type = 'checkbox'>
  • <input type = 'submit'>
  • <input type = 'email'>
  • <input type = 'password'>

We use the <table> tag to make orgainzed charts and data organisers, such as a calendar.

We use the <tr> as number of columns, while the code><td> is the rows in the columns, meaning that they are rows.

Other HTML Codes

For class, it is used to apply css to the applied codes. For id, it also has the same function but the codes are different and the id is the subunit of the class. We use the <class> & <id> tags for the two of them. We either use <h1 class="heading"> or <h3 id="small heading">.

We use two different types of file paths to link to external media such as, images and videos. Relative path is a shorter link, usually in the person's desktop, while the absolute path shows the entire link. We use the <img src='desert.png'> or <iframe src='www.YouTube.com'> tags.

A website would look greater if we use the responsive tag. It is used for media to enlarge or shrink, according to the tab size. We use the <meta name="viewport" content="width=device-width, initial-scale=1.0"> tag to allow responsive. Then, we use the <style="max-width:100%;height:auto;">

Code can also be defined by several codes, which are:

  • <code> tag to define computer code.
  • <kbd> tag to define keyboard input.
  • <samp> tag to define computer output.
  • <var> tag to define a variable.
  • <pre> tag to define preformatted text.
