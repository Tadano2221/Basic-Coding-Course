What is Javascript?

It is a programming language that determines the behaviour of the websites. We can use this to make the website more interactive.

Basics

First, we will learn how to declare a variable. We use the var name = 'Neha'; code to determine a variable.

The let is also used to declare a variable. But, the value is set forever. You can't change it. For example, let's take let x = '0';

A function is the most important part of JavaScript. For that, we will type function example() { alert('Hello!') } . The alert is used to display messages in a browser.

To assign a variable in a function, we usually put the const pi='3.142857';. The value is constant.

An array is useful when you ahve multiple variables. We use the const people = ["Jane", "Sam", "Andy"];

The Js math library provides 12 functions.

For Boolean logic, type like a (5<3). It will show either True or False.

Use the Math.random(); to get a random number.
