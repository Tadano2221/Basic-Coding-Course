Basic Python Concepts

The main parts of python you need to know is input, print, functions, variables, lists, dictionaries, boolean, math, tuples and libraries. The print is the simplest function. We just type print('Hello, world!') and Then, the text appears on the screen. For variables, we just assign a value to a word. take this for example: age = 10. Then, we need to know about functions. Let us make a sample function:
def greet():
    print('Hello!') Then, we just type greet() to use the function. Next is inputs. We use them for quizzes and questions. It is typed like this: quiz = input('What is the largest continent?'). Lists, dictionaries and tuples are storages of variables but they differ in how they act. Dictionaries have keys and values. For example, age = {'Sam': 13}. Lists look like this: numbers = ['21', '22', '23'] Tuples have values which will be set forever. THey cannot be changed. Example: animals = ('turtle', 'cat') For boolean, there are three functions. They look like this:

  • 3 < 5
  • 3 > 5
  • 3 == 5.
Depending on the values, they will display either True or False. We use +, -, * and / for the four mathematical operators. Libraries are chunks of code that you may want to apply them to your codes. Turtle and Tkinter is one of them. We also use Random. PyGame is used for gaming.
